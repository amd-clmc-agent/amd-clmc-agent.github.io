Detailed Description

Floating-base inverse dynamics with contact forces and joint torques optimization based on the IJRR 2013 paper.

Author
Ludovic Righetti The most general cost function is: 0.5*tau^{T}*command_cost_weight_*tau + command_cost_linear_term_^{T}*tau
  • 0.5*lambda^{T}*contact_cost_weight_*lambda + contact_cost_linear_term_^{T}*lambda By default, this solver minimizes the cost function min_{tau} 0.5||tau||^{2} s.t. (dynamics).
