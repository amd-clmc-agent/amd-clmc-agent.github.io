Detailed Description

This class implements the friction cones as quadratique term on the forces and
linear constraints on the torques.
The number of constraint is up to 1 quadratic constraint + 7 linear per
contact. This number vary in fonction of the number of active contact
and if the contact is unilateral.

equations :
scale_qud * (X^T D X + E X + H) <= 0
scale_lin * A X + B <= 0
